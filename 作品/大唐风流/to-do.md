## 📦 角色卡打包说明

完成所有创作内容后，需要使用打包程序将文件整合成 SillyTavern 可导入的角色卡。支持 JSON 和 PNG 两种格式。

### 第一步：编写配置文件

在项目根目录创建一个 YAML 配置文件（例如：`大唐风流.yaml`），内容参考以下格式：

```yaml
# 基础信息
name: 大唐风流
creator: ""
character_version: ""

# 封面图路径（PNG 打包时直接读取）
cover: 封面图/大唐风流.png

# 主要字段映射
# 对话补充等额外设定应该放入 character_book，而不是使用 mes_example
fields:
  description: 作品/大唐风流/角色设定_主角名.xyaml
  personality: ""
  scenario: ""
  first_mes: 作品/大唐风流/开场白.md
  mes_example: ""
  creator_notes: ""
  system_prompt: ""
  post_history_instructions: ""

# 扩展字段
extensions:
  talkativeness: "0.5"
  fav: false
  world: 大唐风流
  status_bar: 作品/大唐风流/状态栏.html  # 可选

# 角色书配置
character_book:
  name: 大唐风流
  entries:
    # 背景设定
    - comment: "背景设定"
      content: 作品/大唐风流/背景设定.xyaml
      enabled: true
      position: before_char
      insertion_order: 1
      depth: 4
      role: 0

    # 玩家角色
    - comment: "玩家角色_{{user}}"
      content: 作品/大唐风流/玩家角色_{{user}}.xyaml
      enabled: true
      position: before_char
      insertion_order: 2
      depth: 4
      role: 0

    # 其他设定条目（可添加多个）
    - comment: "其他设定条目1"
      content: 作品/大唐风流/其他设定1.xyaml
      enabled: true
      position: after_char
      insertion_order: 1
      depth: 4
      role: 0
```

**配置说明：**

- `name`: 角色卡名称
- `fields`: 各个字段对应的文件路径，空字符串 `""` 表示该字段为空
- `extensions.status_bar`: 状态栏HTML文件路径（可选）
- `character_book.entries`: 角色书条目列表
  - `comment`: 条目名称
  - `content`: 条目内容对应的文件路径
  - `position`: 插入位置（`before_char`/`after_char`/`at_depth`）
  - `insertion_order`: 插入顺序（数字越小越靠前）
  - `depth`: 深度值（1-4）
  - `role`: 角色类型（0=系统，1=用户，2=AI）

**参考示例：** `config.example.yaml`

### 第二步：生成 JSON 角色卡

确保已安装 Node.js 和依赖包，然后在项目根目录运行：

```bash
# 首次使用需要安装依赖
npm install

# 生成 JSON 角色卡
node build-card.js 大唐风流.yaml
```

程序会自动读取配置文件，整合所有内容，生成 `完整作品/大唐风流/大唐风流.json` 文件。

### 第三步：准备封面图

封面图路径已在配置文件中指定：

```
cover: 封面图/大唐风流.png
```

请将封面图放入项目中的上述路径，或在 YAML 配置里修改 `cover` 字段。

### 第四步：生成 PNG 角色卡

打包程序会读取 YAML 中的 `cover` 路径作为封面图：

```bash
# 从 YAML 配置文件直接生成 PNG
node build-card-png.js 大唐风流.yaml

# 或从已生成的 JSON 生成 PNG
node build-card-png.js 完整作品/大唐风流/大唐风流.json
```

程序会将角色卡 JSON 嵌入封面图，生成 `完整作品/大唐风流/大唐风流.png` 文件。
